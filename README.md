
# Panduan Upload Arkana Web ke GitHub Pages

1. Buka https://github.com dan login.
2. Klik tombol **New** untuk membuat repository baru.
3. Beri nama repository sesuai keinginan (misalnya: Arkana-Web).
4. Pilih **Public** agar bisa diakses semua orang.
5. Klik **Create Repository**.
6. Upload semua file dari ZIP Arkana Web ini.
7. Pergi ke tab **Settings** → **Pages**.
8. Pada "Source", pilih **main branch** dan folder `/ (root)`.
9. Klik **Save**.
10. Tunggu ±1 menit, lalu website bisa diakses di:
    `https://username.github.io/repository-name`
